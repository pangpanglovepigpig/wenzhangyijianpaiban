# 文章排版工具部署说明

这个项目现在可以作为网页程序部署。部署后，电脑、手机和平板都能通过同一个网址使用；在支持的浏览器里还能添加到桌面，像普通 App 一样打开。

## 推荐方式：部署到 GitHub Pages

项目已经包含 `.github/workflows/deploy.yml`。推送到 GitHub 的 `main` 分支后，GitHub 会自动构建并发布网页。

首次使用时：

1. 在 GitHub 创建一个新仓库。
2. 把本项目推送到这个仓库的 `main` 分支。
3. 进入仓库的 Settings → Pages。
4. Source 选择 GitHub Actions。
5. 等 Actions 运行完成后，就会得到一个公开网址。

如果仓库名是 `文章一键排版`，网址通常类似：

```text
https://你的用户名.github.io/文章一键排版/
```

如果仓库名是 `你的用户名.github.io`，网址就是：

```text
https://你的用户名.github.io/
```

## 其他方式：部署成在线网页

1. 在本机生成发布文件：

   ```bash
   npm run build
   ```

2. 把生成的 `dist` 文件夹上传到任意静态网站平台，例如：

   - Vercel
   - Netlify
   - Cloudflare Pages
   - 阿里云 OSS / 腾讯云 COS / 七牛云静态网站

3. 得到网址后，把网址发给别人即可使用。

手机安装方式：

- iPhone：用 Safari 打开网址，点分享按钮，选择“添加到主屏幕”。
- Android：用 Chrome 打开网址，选择“安装应用”或“添加到主屏幕”。
- 电脑：用 Chrome 或 Edge 打开网址，地址栏右侧通常会出现安装按钮。

## 临时方式：同一 Wi-Fi 内使用

如果只是临时让身边的手机或另一台电脑试用，可以运行：

```bash
npm run dev
```

然后用同一 Wi-Fi 下的设备访问本机局域网地址。这个方式适合临时测试，不适合作为长期使用入口。

## 如果需要真正的安装包

如果希望做成 macOS / Windows 双击安装的软件，可以在这个网页程序外面再套一层桌面壳，比如 Tauri 或 Electron。手机端仍建议保留网页/PWA 方案，因为它最省维护，也最容易分享给别人。
